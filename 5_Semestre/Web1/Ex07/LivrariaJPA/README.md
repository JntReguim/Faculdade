# LivrariaJPA
Aplicação Spring Boot com Spring Data JPA para gestão de livraria.

## Tecnologias utilizadas
- Spring Boot 3.4.6
- Spring Data JPA
- MySQL 
- Maven

## Pré-requisitos
- Java 21
- Maven 3.6.3+
- MySQL 8.0.21+

## Como executar
1. Construa o projeto:
    mvn clean package

2. Execute a aplicação:
    java -jar target/LivrariaJPA-0.0.1-SNAPSHOT.jar
